# 📌 Project Overview

The purpose of this portfolio website is to:

* Present my **professional profile**
* Showcase **projects and technical skills**
* Demonstrate **modern web development techniques**
* Practice **clean semantic HTML and responsive CSS design**

The website is built **without frameworks**, focusing on **core web technologies** to demonstrate fundamental front-end development skills.
